# CLI

## Angular
Install cli
``` bash
npm install -g @angular/cli
```

Create project
``` bash
ng new example-app
```

Start development server
``` bash
ng serve --open
```

Generate component
``` bash
ng generate component MyComponent
```

Build
``` bash
ng build example-app
```

## React


Create project
``` bash
npx create-react-app example-app
```

Start development server
``` bash
npm start
```

## Svelte

Create project
``` bash
npx degit sveltejs/template my-svelte-project
```

To use TypeScript
``` bash
node scripts/setupTypeScript.js
```

Start development server
``` bash
npm run dev
```

## Vue

Install cli
``` bash
npm install -g @vue/cli
```

Start wizard
``` bash
vue create example-app
# To use GUI
vue ui
```
Select options, e.g.
- TypeScript
- Router
- Vuex
- CSS pre-processors


Start development server
``` bash
npm run serve
```

## Vuetify
Add to vue project
``` bash
vue add vuetify
```

## Quasar

Install cli
``` bash
npm install -g @quasar/cli
```

Create project
``` bash
 quasar create example-app
```

Start development server
``` bash
quasar dev
```
